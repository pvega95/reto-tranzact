# TRANZACT-APP
> Este es un app reto desarollada para TRANZACT
## 💻 Pre-requisitos

* Debe de tener instalado `NODE JS`
* Debe de tener instalado `REACT 18`

## 🚀 Instalando AVLA-APP

Para instalar  AVLA-APP, siga estas etapas:

Windows:
```
NPM INSTALL
```

```
NPM RUN START
```
